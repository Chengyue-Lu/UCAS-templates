\begin{abstract}
本模板旨在为中国科学院大学的学生提供一套灵活、规范的课程论文写作方案。它集成了常用的学术写作功能，包括\textbf{代码高亮}、\textbf{数学公式排版}、\textbf{独立封面生成}、\textbf{草稿水印}以及最新的\textbf{中英文模式切换}。使用者可以通过修改 \texttt{UCASPaper} 包的参数，一键切换论文风格，无需手动调整复杂的 \LaTeX 代码。本文档将作为一份综合指南，演示上述所有功能的具体用法。

\end{abstract}

\section{核心功能配置}

\subsection{中英文模式切换}
在引入包时设置 [english=true]，系统会自动将以下标签转换为英文，适合撰写全英文报告或向国际会议投稿：
\begin{itemize}
    \item 摘要标题 $\rightarrow$ \textbf{Abstract}
    \item 关键词前缀 $\rightarrow$ \textbf{Keywords:}
    \item 参考文献标题 $\rightarrow$ \textbf{References}
    \item 图表标题 $\rightarrow$ \textbf{Figure X} / \textbf{Table X}
\end{itemize}

\subsection{灵活的封面与水印}
\begin{itemize}
    \item \textbf{独立封面}：设置 \texttt{coverpage=true}，会自动生成包含校徽大Logo的独立首页，适合期末大作业。
    \item \textbf{水印标记}：设置 \texttt{watermark=true}，页面背景会出现淡灰色的 "UCAS DRAFT" 字样，适合审阅。
\end{itemize}

\section{排版功能演示}

\subsection{优雅的代码插入}
理工科课程论文经常需要展示算法源码。现在你可以直接使用 standard \texttt{lstlisting} 环境，模板已内置了类似 IDE 的高亮配色：

\begin{lstlisting}[language=Python, caption={Python代码示例: Hello UCAS}]
def hello_ucas(name):
    """
    这是一个简单的函数文档字符串
    """
    print(f"Hello, {name} from UCAS!")
    
    # 计算斐波那契数列
    a, b = 0, 1
    for _ in range(10):
        a, b = b, a + b
    return a

if __name__ == "__main__":
    result = hello_ucas("Student")
    print(f"Result: {result}")
\end{lstlisting}
